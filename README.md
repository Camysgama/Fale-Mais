# Fale-Mais-Vizir

### 👍 Rodando a Aplicação

```bash
# Clone este repositório
$ git clone https://github.com/Camysgama/Fale-Mais.git

# Acesse a pasta do projeto 
$ cd Fale-Mais

# Instale as dependências
yarn install

# Execute o projeto
yarn dev
```
